# מדריך להורה — חכמוני

*ידידותי, בעברית, ללא ז'רגון-טכני.*

## איך להתחיל

1. **לחיצה כפולה על האייקון "חכמוני" בשולחן-העבודה** — המשחק נפתח אוטומטית בChrome.
2. אם זו פעם ראשונה — תופיע דמות-המורה (פרופ' חכמוני, ינשוף ידידותי) ותלמד את הילד מה לעשות.

## חוויית פעם-ראשונה (CHG-005)

המשחק מתחיל **בלי משתמשים קבועים**. הילד יוצר את עצמו ואז את ההורים:

### חלק 1 — Fast-Path (~30 שניות)
1. הילד רואה את חכמוני: "שלום! מה השם שלך?"
2. שדה-שם + כפתור 🔊 "שמע את השם" (לוודא שהמשחק יבטא נכון את השם)
3. ברירות-מחדל: גיל=5, צבע=שמש, אווטאר=ארנב
4. תרגול-קטן (לחיצה על כוכב) → ישר למשחק

### חלק 2 — Progressive Onboarding (אחרי 5 משימות מוצלחות)
חכמוני מופיע: "כל הכבוד! עכשיו בוא נכיר אותך טוב יותר ונזמין את אבא/אמא."
1. **עריכת הילד:** גיל (4/5/6/7/8), צבע-מועדף (8 צבעים), אווטאר/תמונה
2. **הזמנת הורה:** הילד לוחץ "📲 קרא לאמא/אבא"

## הגדרת פרופיל-הורה

זה החלק שלך! עד **2 הורים** (אבא + אמא) יכולים להיות חלק מהמשחק.

1. **שם:** "אבא" / "אמא" / שם-פרטי (לדוגמה: "מוטי")
2. **PIN של 4 ספרות:** **תזכור!** זה הקוד שמגן על מסך-ההגדרות.
3. **שאלת-שחזור:** תאריך-הלידה שלך (DD/MM). למקרה ששכחת PIN.
4. **אווטאר או תמונה:** **מומלץ תמונה אמיתית** — הילד יראה אותך ב-cameo בחגיגות.
5. **🎤 הקלטת-קול (4 שניות):** אופציונלית. דוגמה: "אני אוהבת אותך, יואב! כל הכבוד!"
   - תופיע ב-50% מהחגיגות אחרי משימות.
   - הילד שומע את הקול שלך באמת — ערך-רגשי גבוה.

## איפה תופיע במשחק?

ב-4 מקומות:

| איפה | מה הילד רואה / שומע |
|------|---------------------|
| **מסך-פרופיל-הילד** | האווטאר שלך בפינה (קטן, אטימות 60%) — hover מקריא "אבא/אמא" |
| **חגיגת-משימה** | בכל ~2-3 משימות (אקראי 50%) — תמונתך מופיעה ל-3ש' + הקלטה |
| **הקלטת-קול** | אם הקלטת — מתנגנת ההקלטה במקום TTS |
| **Parent Dashboard** | תמונתך בראש מסך-ההגדרות |

## גיבוי

### Export ידני
הגדרות → "Export Data" → קובץ-JSON יורד ל-Downloads. שמור אותו ב-Drive/USB.

### Google Drive Sync אוטומטי
לאחר חיבור-חד-פעמי ל-OAuth (~30 דק', ראה `docs/DEPLOY.md`):
- כל סיום-משימה → גיבוי אוטומטי ל-Drive.
- **הקלטות-קול גם מסונכרנות** (מוצפנות AES-GCM).
- במחשב-חדש: מתחבר → מציע ייבוא.

## איפוס PIN — 3 רמות

אם שכחת PIN:

### רמה 1: שאלת-שחזור
מסך-PIN → "שכחתי" → תקליד DD/MM של תאריך-הלידה שלך.

### רמה 2: מייל-איפוס דרך Drive
אם רמה 1 לא עזרה → "שלחו לי email-איפוס" → גוגל שולח אימות לחשבון-Google המחובר.

### רמה 3: איפוס-מלא (חירום)
URL ידני: `localhost:8080/?reset-pin=family-emergency` → איפוס מלא.

## דיווח על באג

הגדרות → "דווח על באג" → screenshot + תיאור → JSON יורד. שלח את הקובץ ל-developer.

## סקירת התקדמות

הגדרות → "התקדמות הילדים" → טבלה פר-ילד עם כוכבים + זמן-משחק.

## בעיות-נפוצות

ראה `docs/RECOVERY.md` לתרחישי-משבר (PIN שכוח, Drive נמחק, וכו').

## אבטחה ופרטיות

- **כל הנתונים מקומיים** — לא מועלים לשרת-שלנו.
- **הקלטות-קול-הורה מוצפנות** (AES-GCM) ב-IndexedDB וב-Drive.
- **תמונות-פרופיל** מוצפנות גם הן.
- **PIN hashed** ב-PBKDF2 (100,000 iterations + salt) — אבטחה תעשייתית.
- **OAuth scope "drive.file"** בלבד — האפליקציה רואה רק את הקבצים שיצרה.
