# מדריך להורה — חכמוני

*ידידותי, בעברית, ללא ז'רגון-טכני.*

## איך להתחיל

1. **לחיצה כפולה על האייקון "חכמוני" בשולחן-העבודה** — המשחק נפתח.
2. אם זו פעם ראשונה — תופיע דמות-המורה ותלמד את הילד מה לעשות.

## יצירת פרופיל לילד

1. במסך-פתיחה לחץ על "פרופיל חדש" (כרטיס עם +).
2. בחר שם (מהרשימה או הקלד).
3. בחר אווטאר (חיה) או העלה תמונה אמיתית.
4. שמור.

## הגדרת PIN ההורים

במסך-הראשון: לחץ על ⚙ בפינה תחתונה-שמאלית.
- הזן PIN של 4 ספרות (תזכור!).
- אם שכחת — שאלת-שחזור: "תאריך-לידתך (DD/MM)".

## גיבוי

### Export ידני
הגדרות → "Export Data" → קובץ-JSON יורד ל-Downloads.

### Drive Sync אוטומטי
ראה `docs/DEPLOY.md` להגדרה חד-פעמית (~30 דק').

## דיווח על באג

הגדרות → "דווח על באג" → screenshot + תיאור → JSON יורד. שלח את הקובץ ל-developer.

## סקירת התקדמות

הגדרות → "התקדמות הילדים" → טבלה פר-ילד עם כוכבים + זמן-משחק.

## בעיות-נפוצות

ראה `docs/RECOVERY.md` לתרחישי-משבר.
