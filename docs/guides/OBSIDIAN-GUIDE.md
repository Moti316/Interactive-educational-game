# 📘 מדריך Obsidian — לפרויקט חכמוני

> מדריך פשוט להורה. Obsidian הוא "חדר-הבקרה" שלך לפרויקט — דרך נוחה לראות,
> לנווט ולהבין את 35 מסמכי-הפרויקט. הוא **לא** משנה את העבודה של Claude Code —
> הוא כלי-תצוגה בשבילך.

## מה זה vault ואיך פותחים

**vault** = תיקייה שֶ-Obsidian פותח. ה-vault שלך הוא תיקיית-הפרויקט:
`C:\Users\b0066820\Desktop\Claude project\Interactive-educational-game`.

פתיחה: Obsidian → **Open folder as vault** → בחר את התיקייה. ההגדרות
(`.obsidian/`) כבר מוכנות — עברית RTL, graph, backlinks.

## 5 פעולות-יומיום

| פעולה | קיצור | מה זה עושה |
|--------|--------|------------|
| **לוח-בקרה** | פתח `docs/_DASHBOARD.md` | קובץ-הבית — מצב-הפרויקט + קישורים-חמים. **התחל כאן.** |
| **Graph View** | `Ctrl+G` | מפת-קשרים חזותית של כל המסמכים. רואה מי מחובר למי. |
| **קפיצה לקובץ** | `Ctrl+O` | הקלד שם-קובץ → קפיצה מיידית. אין צורך לחפש בעץ. |
| **חיפוש-על** | `Ctrl+Shift+F` | חיפוש טקסט בכל 35 המסמכים בבת-אחת. |
| **Backlinks** | פאנל ימני | פתח מסמך → רואה מי מקשר אליו. שימושי לפני שינויים. |

## איך לקרוא את ה-Canvas

`project-map.canvas` (בשורש) — לוח חזותי של הפרויקט: 3 הצוותים, שלבי-העבודה
(Phases), והמצב הנוכחי. פתח אותו ב-Obsidian — אפשר לגרור, לזום (גלגלת),
ולראות את כל התמונה. הריבועים הצבעוניים מחוברים בקווים שמראים זרימה.

## תגיות — סינון מסמכים

לכל מסמך יש תגית-נושא. בפאנל-התגיות (אייקון `#`) אפשר ללחוץ על תגית ולראות
רק את המסמכים שלה:

| תגית | מה כוללת |
|------|----------|
| `#status` | מצב — ROADMAP, PROGRESS, RISKS, _DASHBOARD |
| `#team` | 3 מסמכי-הצוות |
| `#log` | יומנים — DECISIONS, COUNCIL, ISSUES, CHANGELOG |
| `#process` | פרוטוקולים ותהליכים |
| `#spec` | מפרט ותוכן — ARCHITECTURE, CONTENT, NARRATION |
| `#quality` | אבטחה, ביצועים, בדיקות |
| `#guide` | מדריכים (כולל זה) |

## מה לא לגעת

- `.obsidian/workspace.json` — נשמר אוטומטית, מצב-תצוגה זמני (לא ב-git).
- `.obsidian/app.json` ושאר ההגדרות — Claude Code הכין; שינוי עלול לבטל RTL.
- אל תמחק את `_INDEX.md`, `_DASHBOARD.md`, `project-map.canvas` — הם נקודות-הכניסה.

## טיפ אחרון

Obsidian שומר הכל בקבצי-markdown רגילים — אותם קבצים ש-Claude Code עובד איתם
וש-git שומר. אין "פורמט-Obsidian" נפרד. מה שתראה ב-Obsidian = מה שיש בריפו.
